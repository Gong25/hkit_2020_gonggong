### 3.8 기본형 매개변수와 참조형 매개변수
* 기본형 매개변수
```java
public class Animal {
    int height;
}
```
```java
public class Main {
    
    public static void diet(int arg) {
        arg = arg / 2;
    }
    
    public static void main(String[] args) {
        Animal dog = new Animal();        
        dog.height = 10;
        diet(dog.height);
        System.out.println(dog.height);
    }
}
```
