* IN, ANY, ALL
```
비교 연산자는 연속된 범위값 또는 하나의 값을 판단하는 반면, 
얘네들은 구멍이 쏭쏭 뚫린 뛰엄 뛰엄 값들을 판단함.

IN과 ANY
IN이 점's이라면,  
ANY는 범위's임.

ALL
IN과 ANY가 OR이라면, 
ALL은 AND임.
```
