## 메모리 참조
* 변수 주소 출력하기
```c++
#include <stdio.h>

int main()
{
	int  a;

	a = 10;

	printf("%d \n", a);  // 변수값 출력하기
	printf("%d \n", &a);  // 변수주소 출력하기

	return 0;
}
```
  - 출력 결과
  ```console
  10
  1703712
  ```
* 참조 변수
  - main.cpp
  ```c++
  #include <stdio.h>

  int main()
  {
    int  a;  // 일반 변수
    int* b;  // 참조 변수

    a = 10;
    b = 20;

    printf("%d \n", a);  // a의 변수값 출력하기
    printf("%d \n", &a);  // a의 변수주소 출력하기

    printf("%d \n", b);  // b의 변수값 출력하기
    printf("%d \n", &b);  // b의 변수주소 출력하기

    return 0;
  }
  ```
  - 실행결과
  ```console
  10
  1703712
  20
  1703716
  ```
